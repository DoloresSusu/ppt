# Part 2: 🧭 快速导航

## 👤 我是志愿者，应该看哪部分？

> **使用说明**：找到你的角色，点击"查看 SOP"快速跳转。到场前请务必阅读你的角色 SOP。

| 角色 | 核心职责（一句话） | 到场时间建议 | 查看 SOP |
|------|------------------|------------|---------|

| 🛠️ **技术支持** | 排障、指导工具使用、兜底 | 提前 1.5 小时 | [点击查看](./roles/tech-support) |

| 🎤 **主持人** | 控场、氛围、连接参与者 | 提前 1.5 小时 | [点击查看](./roles/mc) |
| 📁 **作品管理员** | 提交流程、催收、投票统计 | 提前 1 小时 | [点击查看](./roles/submission) |
| 👨‍🏫 **演示官** | 工具演示、教学、答疑 | 提前 1 小时 | [点击查看](./roles/demo) |

## 🚨 快速应急索引

> **说明**：遇到突发状况？在这里快速找到解决方案。详细应急方案已整合在各角色 SOP 中。

| 突发状况 | 负责角色 | 快速跳转 |
|---------|---------| ---------|
| 签到系统崩溃 | 后勤/签到 | [查看应急方案](./roles/logistics#emergency) |
| 网络故障/工具打不开 | 技术支持 | [查看应急方案](./roles/tech-support#emergency) |
| 设备故障（投影/音响） | 现场电脑操作员 | [查看应急方案](./roles/operator#emergency) |
| 环节严重超时 | 主持人 + 计时/场控 | [查看应急方案](./roles/mc#emergency) |
| 参与者找不到场地 | 后勤/签到 | [查看应急方案](./roles/logistics#emergency) |
| 作品提交不上 | 作品管理员 | [查看应急方案](./roles/submission#emergency) |
| 志愿者临时缺席 | 志愿者统筹 + 主理人 | [查看应急方案](./roles/volunteer-lead#emergency) |
| 赞助商临时到访 | 赞助商对接 | [查看应急方案](./roles/sponsor#emergency) |
