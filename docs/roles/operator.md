# 💻 现场电脑操作员 SOP (中控)

> **核心定位**: 视听体验的保障者，路演环节的"交通指挥官"。
> **📝 使用提示**: 请将下方清单视为 Checklist，完成一项勾选一项。



## 一、角色关键职责
*   **投屏 (Screen)**：管理 PPT、视频、演示软件的投屏，确保画面清晰、切换流畅。
*   **音乐 (Audio)**：根据环节播放合适的背景音乐（BGM）。
*   **协作 (Support)**：配合主持人、讲师进行翻页或演示。**与主持人、计时员站在一起。**
*   **路演场控**: 管理路演顺序，确保切换不卡顿。
*   **记录保障**: **确保会议录制正常进行。**



## 二、工作流程清单

### ⏰ 活动前 (12:00 - 14:00)
- [ ] **设备连接**: 确认主控电脑连接投影仪/大屏，画面比例正常，色彩无误。
- [ ] **音频测试**: 确认电脑声音能通过现场音响播出，无爆音、回声。
- [ ] **录制检查**: **务必确认飞书/腾讯会议的录制功能已开启**，并检查云录制空间是否充足。
- [ ] **文件汇总**: 收集所有需要展示的 PPT、视频、Idea 表格、计时器网页，**建议全部拷贝到本地**，防止断网。
- [ ] **BGM 准备**:
    - 暖场/休息：轻快、舒适 (Lo-Fi, Jazz Hiphop)。
    - 开发环节：专注、无歌词 (Alpha Waves, Coding Mode)。
    - 倒计时/紧张：快节奏。
    - 颁奖：激昂、喜庆 (Victory, Celebration)。

### 🚀 活动流程中
- [ ] **暖场**: 播放暖场音乐，循环播放活动海报或社区介绍轮播图。
- [ ] **开场/路演**:
    - 配合主持人节奏翻页 PPT。
    - **Idea Pitch 环节**: 投屏在线 Idea 表格，实时录入/展示。
- [ ] **开发环节**:
    - 播放专注背景音乐（音量适中，不干扰交流）。
    - 投屏倒计时（大大的数字），增强紧迫感。
- [ ] **作品展示**:
    - 确认所有演示者已知道自己的顺序（将顺序表投屏）。
    - **控场技巧**: 如果演示超时，主持人示意后，**慢慢调低音量**（Fade out）作为背景音乐，而不是直接暂停/静音，避免现场气氛骤冷。
    - 投屏投票二维码。
- [ ] **颁奖**: 播放颁奖 BGM，投屏获奖名单/奖状模板。



## 三、路演投屏方案

### 方案 A: 提交作品到指定地点（推荐）
- 提交作品到飞书问卷， 或者社区网站作品提交地址
- 展示的时候按作品提交顺序即可，投票也可以使用相同的技术方案

### 方案 B: 腾讯会议投屏 (推荐 - 适用于 < 10个作品)
为避免现场频繁拔插 HDMI 线导致黑屏或接口不兼容：
1.  **准备**: 提前创建一个**腾讯会议房间**，将会议号发到群里。
2.  **测试**: 要求所有路演者提前加入会议，测试麦克风和屏幕共享。
3.  **执行**:
    - 你的电脑作为**主控**（连接大屏），进入腾讯会议，关闭麦克风/扬声器（防啸叫）。
    - 路演者在自己座位上加入会议 -> 开启屏幕共享 -> 开始讲。
    - 讲完 -> 停止共享 -> 下一位立马接上。

### 方案 C: 摆摊模式 (推荐 - 适用于 > 10个作品)
如果队伍太多，逐一上台讲会超时，采用摆摊模式：
1.  **摊位**: 每个小组固定在一个桌子（摊位）。
2.  **标识**: **必须打印/手写 A4 纸**，写上：
    - **队名**
    - **作品名称**
    - **一句话介绍**
3.  **流动**: 所有人自由走动，去各个摊位围观、交流、投票。



## 四、常见问题与应急处理

| 问题 | 解决方案 |
|------|---------|
| **投影仪无信号/黑屏** | 检查HDMI线连接；重启投影仪；尝试更换转接头或线缆 |
| **音响无声音/啸叫** | 检查音频线连接；调整音量大小；关闭多余的麦克风/扬声器 |
| **PPT/视频打不开** | 使用本地备份文件；或快速下载/转格式；实在不行跳过该环节 |
| **录制中断/失败** | 立即重新开启录制；检查存储空间；使用多设备同时录制作为备份 |
| **腾讯会议投屏卡顿** | 切换到本地文件投屏；或引导演示者使用HDMI线连接 |



## 五、操作员锦囊
1.  **静音**: 自己的微信/钉钉**务必退出或静音**，避免弹窗尴尬。
2.  **防啸叫**: 腾讯会议模式下，**现场只保留一套音响系统开启**（通常是主控电脑连接的音响），其他人的电脑必须静音且关闭麦克风。
3.  **录制**: 活动开始前设置闹钟提醒自己开启录制，中途最好检查一次是否中断（如网络波动导致录制停止）。
