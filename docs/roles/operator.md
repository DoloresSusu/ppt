# 💻 现场中控 SOP

**一句话目标**：稳定投屏与音频灯光，控时控序，确保流程顺畅。

小型活动中控可兼任时间官。

## 现场执行清单（活动当天）

### 活动前一天
- [ ] 收集 PPT、视频、灵感表格、计时器链接，统一拷贝本地。
- [ ] 确定作品收集方式（官网下载/提交表/U 盘），预设本地文件夹。
- [ ] 准备 BGM 清单（暖场/开发/颁奖）。
- [ ] 准备投票榜单/排行榜展示页，确认可大屏显示。
- [ ] 确认录制方式与账号（飞书/腾讯会议），预留存储空间。
- [ ] 准备倒计时道具（屏幕计时/实体牌）与投票二维码素材。

### 12:00 - 14:00 | 设备联调
- [ ] 连接投影/大屏，画面比例与色彩正常。
- [ ] 测试音频输出，避免啸叫。
- [ ] 测试浏览器与播放器，确认 MP4 可全屏播放。
- [ ] 如有灯光方案，按开场/展示/颁奖预设。
- [ ] 检查转接头与备用线材。
- [ ] 与主持人对齐计时信号与报时节点。
- [ ] 预备备用热点或场地网络兜底。

### 活动流程中
- [ ] 暖场播放海报轮播与背景音乐。
- [ ] 开场/路演按主持节奏翻页或切换素材。
- [ ] 开发阶段投屏倒计时，音乐音量适中。
- [ ] 关键节点提前 5 分钟提醒主持/分享者。
- [ ] 作品展示时投屏顺序表与投票二维码（投票窗口固定 15-20 秒）。

### 16:30 - 17:00 | 作品检查与导出
- [ ] 与技术导师核对提交名单，统一下载作品到本地。
- [ ] 抽样试播音视频，发现问题立即回退或提示改格式。
- [ ] 必要时用 U 盘收集，避免依赖现场网络。
- [ ] 预设展示顺序与计时提示（建议 60-90 秒高光）。

### 活动后
- [ ] 导出录制文件与展示素材，交给主理人/宣发。

## 注意事项与应急

### 关键标准
- 所有核心文件本地备份，防断网。
- 录制开启后 30 分钟自检一次，避免中断。
- 音量宁小勿大，确保可交流。
- 展示节奏优先，长作品仅播放高光片段。

::: tip 大型活动可拆分
可拆为投屏/PPT、中控音乐灯光、计时与顺序管理三岗。
:::

### 路演投屏方案速选
- **本地投屏**：收集作品文件后本机播放，最稳定。
- **会议投屏**：腾讯会议投屏，适合 < 10 队伍。
- **摆摊模式**：队伍 > 10 时采用摆摊式展示，减少主屏负担。

### 常见问题与应急处理
| 问题 | 解决方案 |
| --- | --- |
| **投影无信号** | 检查线材/转接头，重启投影，必要时切换设备 |
| **音响啸叫** | 关闭多余麦克风，拉开音响与麦克风距离 |
| **PPT/视频打不开** | 切换本地备份或转 PDF，应急跳过 |
| **录制中断** | 立即重新开启，必要时用手机/备用设备录制 |
| **作品卡顿/无法播放** | 切换本地下载版本或 U 盘备份 |
