# 📝 签到接待 SOP (进阶)

> 适用于 50 人以上的大型活动，从后勤组中独立拆分。

## 一、工作职责
*   **高效核销**: 确保 50+ 人在 30 分钟内快速入场，不造成拥堵。
*   **第一印象**: 提供热情、专业的接待体验。
*   **人流分流**: 引导不同身份（参与者、嘉宾、观众）快速找到位置。

## 二、工作流程

### ⏰ 签到前 (13:00 - 13:30)
- [ ] **签到台布置**:
    - 设置在入口显眼处，不阻挡通道。
    - 摆放物料：签到表（按首字母排序）、姓名贴、马克笔、活动指南。
    - 张贴二维码：入群码、WiFi 码。
- [ ] **分工 (建议 2 人)**:
    - **A 角**: 负责扫码/核对名单。
    - **B 角**: 负责发放物料、指引写名牌。

### 🚀 签到高峰 (13:30 - 14:00)
- [ ] **快速核销**:
    - 询问姓名/手机号后四位 -> 勾选名单。
    - 未报名者：引导至旁边的“等待区”，待开场后视空位情况放入。
- [ ] **物料发放**:
    - 递给参与者贴纸和笔，**盯着他/她写完名字并贴在胸前**。
    - 贴上对应的“技能身份贴”（如：红色=程序）。
- [ ] **引导入座**:
    - 指引方向：“往里面走，找个空位坐下，WiFi 密码在群里。”

### 🏁 迟到接待 (14:00 - 14:30)
- [ ] **留守**: 至少留 1 人在签到台等待迟到者。
- [ ] **静默入场**: 引导迟到者从后门或侧边悄声入场，不打扰主持人。

## 三、常见问题
| 问题 | 解决方案 |
| :--- | :--- |
| **未报名空降** | 先请他在旁等候，确认已报名者都入场后，如有空位再安排。 |
| **找不到名字** | 询问是否使用了昵称？查看手机号后四位；如确认报名成功截图，可手动补录。 |
| **排队太长** | 立即增开一条通道（如使用纸质表同时核销）。 |

## 四、锦囊
1.  **笔要多**: 签到台至少准备 5-10 支粗头马克笔，笔最容易丢。
2.  **大声引导**: 高峰期人多嘈杂，要敢于大声引导：“报名的这边排队！写好名字的往里走！”
