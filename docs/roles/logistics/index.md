# 📦 物料/场地 SOP

**一句话目标**：搞定场地、物料和复原，保障动线通畅与现场安全。

## 现场执行清单（活动当天）

### 活动前一天
- [ ] 核对物料清单（签到表、姓名贴、笔、投票筹码、A4 纸、排插）。
- [ ] 确认场地地址与楼层，更新群公告的“如何到达”。
- [ ] 准备/打印签到表与 WiFi 密码牌。

### 12:00 - 13:30 | 场地布置（必须在 13:30 前完成）
- [ ] 张贴指引牌（楼下大堂/电梯口/转角）。
- [ ] 桌椅按分组岛屿摆放，留出走动通道。
- [ ] 铺设排插到各岛屿，**胶带固定地面线缆**。
- [ ] 搭建签到台（签到表、笔、姓名贴、投票筹码）。
- [ ] WiFi 密码牌放在显眼处。

### 14:00 - 17:30 | 机动保障
- [ ] 入口动线保持通畅，及时移走阻塞物。
- [ ] 物料不足时补给（姓名贴、笔、投票筹码）。
- [ ] 协助主持人/签到处理临时物料需求。

### 17:00 | 投票环节
- [ ] 发放/回收投票筹码。
- [ ] 协助计票，提前确认计票方式。

### 18:00 | 清场复原（必做）
- [ ] 桌椅归位，排插收纳。
- [ ] 垃圾打包带走（瓶子/纸屑）。
- [ ] 检查遗失物，拍照发群。

## 注意事项与应急

::: info 角色核心目标
你是现场的“后勤中枢”，负责物料与场地保障，让活动有序、可靠、可交付。
:::

物料/场地负责物料准备、场地布置和清场复原，并在活动中提供机动支持。

::: warning 核心任务
活动结束后必须还原场地，确保社区信誉，方便下次借用。
:::

::: tip 协作岗位
大型活动建议单独安排签到接待，专注入场核销与人流引导：
*   **[签到接待 SOP](./check-in)**：专职负责入场体验与人流引导。
:::

### 常见问题与应急处理

| 问题 | 解决方案 |
|------|---------|
| **有人找不到场地** | 提前安排一位志愿者在楼下/地铁口接应，或通过微信/电话远程指引 |
| **物料不够（如姓名贴、笔）** | 向附近便利店应急采购，或向场地方借用 |
| **排插不够/跳闸** | 检查是否有大功率设备（如电热水壶），分散用电负荷 |
| **场地空调/灯光故障** | 立即联系场地方物业，同时告知主理人调整流程 |

### 物料/场地锦囊
1.  **动线优先**: 签到台与通道不要堵，入口保持清晰。
2.  **备用物料**: 多备几支粗头马克笔和胶带，现场消耗最快。
3.  **人肉指引**: 高峰期安排一位志愿者在楼下接人，体验最佳。
