# 📦 物料管理 SOP (进阶)

> 适用于 50 人以上的大型活动，从后勤组中独立拆分。

## 一、标准物料清单
请务必在活动前与 **[活动主理人](/organizer/)** 核对最终的物料需求。

### 🏢 场地与设备类
- [ ] **插线板**: 至少 8-10 个，延长线 3-5 根。
- [ ] **转接头**: HDMI、Type-C 扩展坞（至少 3 个备用）。
- [ ] **翻页笔**: 2 支（备用电池）。
- [ ] **麦克风电池**: 备用 5 号/7 号电池 2 板。

### 📝 文具与标识类
- [ ] **签到表**: 打印版 4 份 + 签字笔 5 支。
- [ ] **姓名贴**: 空白不干胶贴纸 (大号) x 2 包。
- [ ] **身份贴**: 彩色圆点贴纸 (红/黄/蓝) 用于区分技能。
- [ ] **A4 纸**: 2 包。
- [ ] **投票物料**: 扑克牌/游戏币 (参与人数 x 3)。
- [ ] **垃圾袋**: 大号垃圾袋 2 卷。

###  奖品与氛围类
- [ ] **饮用水**: 矿泉水 3-4 箱，或大桶水 + 纸杯。
- [ ] **零食**: 独立包装小零食。
- [ ] **奖品**: 确认一二三等奖到位。
- [ ] **证书**: 打印好的荣誉证书。

## 二、物料管理流程

### 📅 筹备期
- [ ] **采购**: 根据预算采购物资，**保留好所有票据**。
- [ ] **领用**: 提前从社区仓库领用公用物资（插排、转接头）。
- [ ] **打包**: 按功能区打包（如“签到包”、“设备包”），贴上标签。

### ⏰ 活动现场
- [ ] **布场**: 协助后勤组铺设插排，将签到物资移交签到组。
- [ ] **巡查**: 随时关注饮用水、纸巾消耗，及时补充。
- [ ] **保管**: 重点看管转接头、翻页笔等易丢小物件。

### 🏁 活动结束
- [ ] **回收**: 必须回收所有插排、转接头。
- [ ] **盘点**: 记录消耗量和遗失情况，向主理人汇报。
- [ ] **归库**: 将公用物资放回指定位置。

## 三、锦囊
1.  **胶带是神器**: 宽胶带（封箱）和美纹纸（不留胶）各备一卷。
2.  **应急包**: 准备创可贴、消毒湿巾、润喉糖。
3.  **防丢**: 贵重物品（如转接头）要专门盯着，活动结束立刻回收。
