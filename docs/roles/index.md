# 👥 岗位角色 SOP

这里包含了周周黑客松活动中各个岗位的详细执行标准作业程序（SOP）。

## 统筹与管理
- [活动主理人 (Host)](./host) - 总统筹、决策、志愿者管理
- [志愿者统筹 (Volunteer Lead)](./volunteer-lead) - 招募、排班、后勤保障
- [鼓励师/顾问 (Advisor)](./advisor) - 关怀志愿者、提供建议

## 内容与流程
- [主持人 (MC)](./mc) - 控场、氛围、连接参与者
- [计时/场控 (Timekeeper)](./timekeeper) - 严格计时、提醒超时
- [分享嘉宾 (Speaker)](./speaker) - 内容输出、预演、控时
- [讲师 (Demo)](./demo) - 工具演示、教学、答疑

## 现场与技术
- [现场电脑操作员 (Operator)](./operator) - 投屏、音乐、路演管理
- [技术支持 (Tech Support)](./tech-support) - 排障、指导工具使用、兜底
- [作品管理员 (Submission)](./submission) - 提交流程、催收、投票统计

## 后勤与支持
- [后勤/签到 (Logistics)](./logistics) - 签到核销、场地布置、清场
- [餐饮与后勤 (Catering)](./catering) - 订餐、分发、垃圾清理
- [物料员 (Material)](./material) - 物资管理、分发、回收

## 宣传与影像
- [宣发推送 (PR)](./pr) - 海报、推文、现场直播
- [摄影 (Photo)](./photo) - 拍照、录像、素材输出
- [赞助商对接 (Sponsor)](./sponsor) - 权益兑现、接待、拍照
