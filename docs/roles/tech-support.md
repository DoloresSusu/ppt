# 🛠️ 技术支持 SOP

> **核心定位**: 现场的"救火队员"，技术难题的"终结者"。
> **📝 使用提示**: 请将下方清单视为 Checklist，完成一项勾选一项。



## 一、角色关键职责
*   **排障 (Troubleshoot)**：解决现场网络、账号、环境配置等基础问题。
*   **指导 (Mentor)**：提供工具使用建议，优化 Prompt，辅助剪辑。
*   **兜底 (Support)**：当现场 WiFi 崩溃时，提供备用热点方案。



## 二、能力要求
*   熟练掌握活动相关 AI 工具（如：Cursor, Suno, 剪映等等）。
*   了解基础的网络配置（DNS, 代理等，视活动需求而定）。



## 三、工作流程清单

### ⏰ 活动前
- [ ] **测试**: 提前测试现场网络环境，确认常用 AI 工具网站能否流畅访问。
- [ ] **账号**: 协助整理/分发公用账号（如有）。
- [ ] **网络准备 (重要)**:
    - 不是所有人都会魔法网络，建议准备国内可用的镜像站/替代工具。
    - 准备备用的本地工具包或安装包。

### 🚀 开发环节 (核心工作区)
- [ ] **主动巡场**: 不要等别人举手。看到有人对着屏幕皱眉、发呆超过 5 分钟，主动凑过去问："遇到什么卡点了吗？"
- [ ] **常见问题处理**:
    - "注册不了/收不到验证码" -> 推荐备用工具或借用公用号。
    - "生成效果不好" -> 帮忙优化 Prompt，或推荐参考案例。
    - "视频做不出来" -> 建议用 PPT + 转场 + 配音 快速替代。
    - "断网了" -> 指导开启手机热点。

### 🏁 作品提交环节 (16:50 - 17:00)
- [ ] **提交协助**: 重点关注那些"做完了但不会交"的队伍，指导他们上传截图/填表。
- [ ] **格式检查**: 确认提交的链接有效（开启了分享权限）。



## 四、常见问题与应急处理

| 问题 | 解决方案 |
|------|---------|
| **现场 WiFi 崩溃** | 指导参与者开启手机热点，或联系场地方重启路由器 |
| **AI 工具无法访问** | 推荐国内镜像站/替代工具，或提供科学上网临时方案 |
| **工具账号用尽额度** | 使用备用账号池，或引导参与者注册新账号 |
| **电脑环境配置失败** | 提供在线版工具替代本地安装，或帮助远程调试 |
| **大面积技术问题** | 立即告知主理人和主持人，考虑调整活动流程或延长时间 |



## 五、技术支持锦囊
1.  **授人以渔**: 教他们怎么写 Prompt，而不是替他们写。
2.  **降级方案**: 如果某个工具挂了，立刻推荐替代品（Plan B）。
3.  **心态安抚**: 很多时候参与者焦虑是因为工具报错，告诉他们"这很正常，我们换个路子"。
