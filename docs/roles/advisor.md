# 🧙‍♂️ 鼓励师/顾问工作 SOP

## 一、角色关键职责
*   **顾问 (Advisor)**：作为主理人的"外脑"，在活动筹备和突发状况时提供决策建议。
*   **激励 (Cheerleader)**：时刻关注**志愿者**的状态（累不累？委不委屈？），给予及时的肯定和情绪价值。
*   **兜底 (Backup)**：当现场出现主理人无法搞定的"硬骨头"时，利用你的经验出面化解。



## 二、工作流程清单

### 📅 筹备期 (作为军师)
- [ ] **流程诊断**: 帮主理人过一遍流程表，指出那些"想当然"的漏洞（比如：这点时间够不够大家吃披萨？）。
- [ ] **心理按摩**: 新手主理人容易焦虑，你要告诉他/她："别怕，有我在，出事我帮你兜着。"

### ⏰ 活动当天 (作为啦啦队长)
- [ ] **隐形观察**: 站在后排，不轻易插手具体执行，除非流程严重跑偏。
- [ ] **关怀志愿者 (重点!)**:
    - 看到签到的小伙伴站久了，递瓶水。
    - 看到技术小哥被小白缠住很烦躁，过去拍拍肩解围。
    - 看到摄影师满头大汗，夸一句"刚才那个角度绝了"。
- [ ] **记录高光**: 随时记录志愿者们的感人瞬间（用手机备忘录记下来，这是致谢环节的弹药）。

### 🏆 致谢环节 (17:55 - 高潮时刻)
- [ ] **配合致谢**: 站在主理人身边，为志愿者颁发证书。
- [ ] **公开夸夸 (你的主场)**:
    - **拒绝套路**: 不要说"大家都很棒"。
    - **必须走心**: 拿着你的小本本，点名表扬！
    - *话术示例*: "我想特别感谢@小明，今天只有他注意到了厕所没纸，自己跑去买了；还有@小红，为了帮那个队伍调代码，午饭都没吃。这才是我们的黑客松精神！"



## 三、常见问题与应急处理

| 问题 | 解决方案 |
|------|---------|
| **志愿者情绪低落** | 私下了解原因，给予肯定和鼓励；必要时调整其工作内容或安排休息 |
| **主理人决策犹豫** | 提供2-3个可行方案供其选择，必要时帮助做最终决策 |
| **现场出现严重冲突** | 立即介入调解，必要时请冲突方私下沟通；保护志愿者不受委屈 |
| **活动严重跑偏** | 与主理人商议，果断调整流程；必要时亲自上阵救场 |



## 四、顾问锦囊
1.  **不仅是对外，更是对内**: 大家都关注参与者开不开心，**你只关注志愿者开不开心**。志愿者开心了，服务质量自然就好了。
2.  **少指挥，多建议**: 把光环留给主理人，把建议私下告诉他。
3.  **看到被忽视的角落**: 那些默默无闻搬桌子、扫垃圾的人，才是你最该大声表扬的人。
