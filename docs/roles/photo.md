# 🎥 摄影 SOP


摄影负责记录活动过程，产出高质量照片/视频，并在活动当晚输出精选素材，重点捕捉专注、协作与情绪瞬间。



## 一、拍摄清单

### 📸 照片
1.  **空镜 (3-5张)**: 场地布置、易拉宝、奖品摆台（活动开始前拍）。
2.  **签到 (5-10张)**: 签到瞬间、领物料、贴名牌。
3.  **全景 (3-5张)**: 开场时座无虚席的画面，大家聚精会神听讲的画面。
4.  **特写 (重点!)**:
    - **专注**: 参与者盯着屏幕思考、敲代码的侧颜。
    - **协作**: 两个人指着屏幕讨论、画白板、比划手势。
    - **情绪**: 开怀大笑、皱眉思考、击掌庆祝。
5.  **路演**: 每个队伍展示时的画面（尽量带上投影屏幕）。
6.  **颁奖**: 获奖者与颁奖人合影。
7.  **大合影**: 全员合影（确保没人被遮挡，喊口号）。

### 📹 视频
- [ ] **花絮**: 捕捉动态瞬间（签到、讨论、展示），用于剪辑 30s 回顾短片。
- [ ] **采访 (可选)**: 随机采访 1-2 人："今天感觉怎么样？" "最喜欢哪个环节？"



## 二、工作流程清单

### ⏰ 活动中
- [ ] **多角度**: 蹲下拍、站高拍、侧面拍，不要只用视平线视角。
- [ ] **避雷**: 尽量避开闭眼、表情崩坏、驼背的画面。
- [ ] **互传**: 拍到好照片，可以实时发几张到活动群，活跃气氛。

### 🏁 活动后 (24小时内)
- [ ] **筛选**: 剔除废片，精选 30-50 张。
- [ ] **修图**: 简单调色（亮度、对比度、白平衡），统一色调。
- [ ] **上传**: 上传至百度网盘/相册链接/社区网站对应的活动相册，然后发布到活动群聊中

## 四、常见问题与应急处理

| 问题 | 解决方案 |
|------|---------|
| **光线不足/逆光** | 调整拍摄角度避免逆光；使用相机/手机的HDR模式；必要时借用现场灯光 |
| **设备电量/存储不足** | 提前准备备用电池和存储卡；活动中定期检查设备状态 |
| **关键瞬间错过** | 提前与主持人确认关键时间点（如颁奖时间）；在重要环节前提前就位 |



## 四、摄影锦囊
1.  **横构图**: 适合公众号推文；**竖构图**: 适合发视频号/朋友圈/抖音。建议两者兼顾。
2.  **光线**: 尽量顺光拍摄，逆光时注意补光或调整角度。
3.  **打扰**: 拍摄时尽量"隐形"，使用长焦或静音快门，不要打断参与者的思路。
