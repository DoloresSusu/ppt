# 🤝 志愿者统筹 SOP (Volunteer Lead)

> **核心定位**: 志愿者的"HR"与"大管家"，确保志愿者团队本身运转流畅。
> **一句话**: 主理人管活动，你管人；让每一位志愿者都知道自己该干嘛，并且有饭吃。
> **📝 使用提示**: 请将下方清单视为 Checklist，完成一项勾选一项。



## 一、角色关键职责
*   **招募与排班 (Recruit & Schedule)**：确认谁能来，谁干什么，什么时间到。
*   **现场调度 (Dispatch)**：活动现场发现哪个岗位缺人，立即调配机动人员补位。
*   **后勤保障 (Welfare)**：**最重要的工作之一** —— 确保志愿者有水喝、有饭吃（订餐）。
*   **培训 (Training)**：活动前 30 分钟，召集所有志愿者开简短的 Briefing（碰头会）。



## 二、工作流程清单

### 📅 活动前 (T-7 ~ T-1)
- [ ] **招募**: 在志愿者群发布招募令，确认名单。
- [ ] **分工**: 填写《志愿者分工表》，明确每个人的角色（谁是主持，谁是签到）。
- [ ] **拉群**: 组建当期活动的工作群，发布分工表和 SOP 文档。
- [ ] **订餐**: 统计志愿者人数和忌口（素食/过敏），预订工作餐/奶茶。

### ⏰ 活动当天
- [ ] **集合 (T-2h)**: 点名，确认所有志愿者到齐。如果有迟到的，立即联系或安排替补。
- [ ] **岗前培训 (T-1.5h)**:
    - 召集大家围成一圈。
    - 快速过一遍流程："今天 14:00 开始，17:00 结束。"
    - 强调重点："大家都记得看一眼自己的 SOP 清单，特别是签到组，一定要引导大家扫码入群。"
- [ ] **巡场**:
    - 观察是否有志愿者在"摸鱼"或不知所措，过去指派具体任务。
    - 观察是否有志愿者太累（如一直站着的签到），安排轮换休息。
- [ ] **发饭**: 确保大家在忙碌的间隙能吃上一口热乎饭/喝上水。

### 🏁 活动后
- [ ] **合影**: 组织志愿者单独大合影。
- [ ] **复盘**: 收集大家的反馈，记录在案。



## 三、常见问题与应急处理 

| 问题 | 解决方案 |
|------|---------|
| **志愿者临时缺席** | 立即启用备选方案：安排其他志愿者兼任，或主理人/统筹自己顶上 |
| **志愿者不熟悉工作内容** | 现场进行简短培训（5分钟）；或安排经验丰富的志愿者带领 |
| **志愿者体力不支** | 安排轮换休息；提供座位和饮用水；必要时调整其工作内容 |
| **志愿者之间协作不畅** | 立即介入协调，明确各自职责边界；必要时调整分工 |
| **订餐出错（数量/口味）** | 应急采购补充；或协调志愿者之间调换；记录教训避免下次再犯 |



## 四、统筹锦囊
1.  **把人当人**: 志愿者不是机器，他们也需要休息和被尊重。你的一句"辛苦了，快去喝口水"比什么都强。
2.  **预案**: 永远假设有 1-2 个志愿者临时鸽子，准备好 Plan B（谁能顶替关键岗位）。
3.  **清晰指令**: 不要说"大家去帮忙"，要说"@小明 去搬桌子，@小红 去发水"。
