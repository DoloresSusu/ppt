# 💰 赞助商对接 SOP (Sponsor Liaison)

> **核心定位**: 金主爸爸的"专属客服"，权益兑现的"执行官"。
> **一句话**: 拿了钱/资源，就要把事办漂亮，让赞助商觉得"这就值了"。
> **📝 使用提示**: 请将下方清单视为 Checklist，完成一项勾选一项。



## 一、角色关键职责
*   **权益兑现 (Deliverables)**：确保赞助商答应的 Logo 露出了，展位摆好了，口播念到了。
*   **接待 (Reception)**：负责赞助商代表的迎送、引导入座、茶水服务。
*   **物料管理 (Merch)**：管理赞助商提供的奖品、贴纸、易拉宝等物资。



## 二、工作流程清单

### 📅 活动前
- [ ] **权益确认**: 对照《赞助协议》，列出权益清单 (Checklist)。
    - [ ] Logo 是否在海报/大屏 PPT 上？
    - [ ] 是否有 5 分钟宣讲时间？
    - [ ] 是否需要摆放易拉宝/展位？
- [ ] **物料对接**: 联系赞助商寄送物料（奖品、周边），确认收货。

### ⏰ 活动当天
- [ ] **展位布置**: 提前帮赞助商把桌子摆好，把易拉宝支棱起来。
- [ ] **接待**:
    - 赞助商代表到达时，亲自去门口迎接。
    - 引导至前排 VIP 座位（贴好"贵宾席"贴纸）。
- [ ] **流程监控**:
    - 主持人念口播时，竖起耳朵听，确认没有念错品牌名。
    - 抽奖环节，确认奖品已摆在台上。
- [ ] **拍照**: **极其重要！** 专门叮嘱摄影师："多拍几张赞助商 Logo 的特写，多拍几张赞助商代表颁奖的照片。"（这是给金主交作业的关键）。

### 🏁 活动后
- [ ] **结案报告**:
    - 收集所有含 Logo 的照片/视频。
    - 整理活动数据（多少人参与，群里多少条讨论）。
    - 打包发给赞助商，并附上一句真诚的感谢。



## 三、常见问题与应急处理

| 问题 | 解决方案 |
|------|---------|
| **赞助商临时到访（未提前通知）** | 立即安排座位和接待；临时调整流程给予简短介绍时间 |
| **Logo露出错误/缺失** | 立即修正PPT或海报；如无法修正，口头致歉并承诺后续补偿 |
| **赞助商物料未到场** | 联系物流确认情况；如无法及时送达，使用替代方案（如电子展示） |
| **赞助商代表不满意** | 立即沟通了解诉求；现场能解决的立即解决；无法解决的承诺后续改进 |



## 四、对接锦囊
1.  **品牌即生命**: 绝对不要把赞助商的名字写错、Logo 变形。这是大忌。
2.  **换位思考**: 赞助商也是来做 KPI 的，尽量帮他们多在群里吆喝两句，让他们觉得社区很活跃。
3.  **多拿一点**: 赞助商通常有很多滞销的周边（贴纸、T恤），多要一点发给志愿者当福利，他们通常很乐意。
