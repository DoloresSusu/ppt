# 📦 物料员 SOP

::: info 角色核心目标
你是活动的“后勤大管家”。你的任务是确保活动所需的所有物料都准确无误地准备、分发和回收，为活动的顺利进行提供坚实的物资保障。
:::

物料员需要根据活动策划案中的物料清单，负责物料的采购、领用、现场管理及活动后回收盘点。

## 一、标准物料清单

这是一个通用清单模板，请务必在活动前与 **[活动主理人](/organizer/)** 核对最终的物料需求。

### 🏢 场地与设备类
- [ ] **插线板**: 至少 5-8 个（视人数和场地情况而定），延长线 2-3 根。
- [ ] **转接头**: HDMI、Type-C 扩展坞（至少 2 个备用）。
- [ ] **翻页笔**: 1 支（提前检查电池电量）。
- [ ] **麦克风电池**: 备用 5 号/7 号电池 1-2 板。

### 📝 文具与标识类
- [ ] **签到表**: 打印版 2 份 + 签字笔 3 支。
- [ ] **姓名贴**:
    - 空白不干胶贴纸 (大号) x 1 包。
    - 粗头马克笔 (黑色) x 5 支。
- [ ] **身份贴**: 彩色圆点贴纸（如：红=程序，黄=设计，蓝=产品），用于快速区分技能。
- [ ] **A4 纸**: 1 包（用于临时引导、团队记录等）。
- [ ] **投票物料**: 扑克牌、游戏币或小贴纸（建议数量为 `参与人数 x 3`）。
- [ ] **垃圾袋**: 大号垃圾袋 1 卷。

### � 奖品与氛围类
- [ ] **饮用水**: 矿泉水 1-2 箱，或大桶水 + 纸杯。
- [ ] **零食**: 独立包装、方便取食的小饼干、糖果等。
- [ ] **奖品**:
    - 一等奖 (如社区 T 恤/马克杯)。
    - 二等奖 (如技术书籍)。
    - 三等奖 (如贴纸/徽章)。
- [ ] **奖状/证书**: 打印好的荣誉证书（获奖信息留空）。

## 二、工作流程清单

### 📅 活动前
- [ ] **获取并核对清单**: 主动找 **[活动主理人](/organizer/)** 获取最终版的 **[物料清单](/organizer/planning/#_4-物料清单)**，仔细核对种类、数量和规格。
- [ ] **物料准备**:
    - **采购**: 对于需新购的物料（如零食、电池），根据预算采购，并 **保留好所有票据** 用于报销。
    - **领用**: 对于社区已有物料（如插线板、转接头），联系主理人确认存放位置并提前领用。
- [ ] **清点与打包**: 活动前一天，将所有物料集中，按清单逐一清点。建议分类打包并贴上标签（如“签到台用”、“设备类”），方便现场快速查找。

### 🏃 活动中
- [ ] **现场布置与交接**:
    - 到达现场后，根据场地规划布置物料。
    - 将插线板、延长线铺设到各个小组位置。
    - 将签到用品（签到表、姓名贴、马克笔）交给 **[后勤/签到人员](./logistics)**。
    - 将技术设备（转接头、翻页笔）交给 **[现场中控](./operator)**。
    - 将奖品和证书交给 **[主持人](./mc)** 或主理人。
- [ ] **现场管理**:
    - 保管好备用物料和个人贵重物品。
    - 留意消耗品（如饮用水、纸巾、零食）的使用情况，及时补充。

### 🏁 活动后
- [ ] **物料回收**: 活动结束后，主动从各处回收可重复使用的物料（特别是插线板、转接头、翻页笔等贵重物品）。
- [ ] **清点与归库**: 按照物料清单，清点回收的物资，记录消耗、遗失或损坏的物品。将社区公共物资放回指定存储位置。
- [ ] **报销与同步**:
    - 整理本次采购的票据，提交给主理人进行报销。
    - 在志愿者群里同步物料盘点结果。

## 三、沟通与协作

| 遇到以下情况 | 请联系 |
| :--- | :--- |
| **物料清单不明确或有疑问** | [活动主理人](/organizer/) |
| **预算不足或需要紧急采购** | [活动主理人](/organizer/) |
| **现场物料被挪用或找不到** | 现场所有志愿者 |

## 四、常见问题与应急处理

| 问题 | 解决方案 |
|------|---------|
| **关键物料遗忘或缺失** | 立即告知主理人，并迅速决策：就近购买、向场地方借用、或寻找替代方案。 |
| **插线板不够/跳闸** | 优先向场地方求助；若无法解决，则引导参与者轮流充电，或使用备用充电宝。 |
| **贵重物品丢失（如转接头）** | 立即在活动群发布寻物启事，并小范围询问最后接触该物品的人员。 |
| **采购费用可能超支** | 对于超支部分，必须 **提前** 向主理人报备并获得同意，否则可能无法报销。 |

## 五、物料员锦囊
1.  **胶带是神器**: 常备一卷宽胶带（封箱胶）和一卷美纹纸（不留胶），用于现场固定线缆、张贴标识。
2.  **准备应急包**: 准备一个小的应急包，内含创可贴、消毒湿巾、润喉糖等，以备不时之需。
3.  **失物招领**: 提醒大家保管好个人物品。活动结束后，如发现遗留物品（充电器、水杯等），集中保管并在群内发布失物招领信息。
