# 📦 物料员 SOP


## 一、角色关键职责
*   **准备**：根据清单采购或整理所有活动所需物资。
*   **管理**：现场物资的分发、补给与回收。
*   **盘点**：活动结束后的物资清点与损耗记录。



## 二、标准物料清单

### 🏢 场地与设备类
- [ ] **插线板**: 至少 5-8 个（视人数而定），延长线 2-3 根。
- [ ] **转接头**: HDMI、Type-C 扩展坞（至少 2 个备用）。
- [ ] **翻页笔**: 1 支（检查电池）。
- [ ] **麦克风电池**: 备用 5 号/7 号电池 1 板。

### 📝 文具与标识类
- [ ] **签到表**: 打印版 2 份 + 签字笔 3 支。
- [ ] **姓名贴**:
    - 空白不干胶贴纸 (大号) x 100 张。
    - 粗头马克笔 (黑) x 5 支（写名字用）。
- [ ] **身份贴**: 彩色圆点贴纸（红=程序，黄=设计，蓝=产品）用于区分技能。
- [ ] **A4 纸**: 1 包（用于临时写队名、草稿）。
- [ ] **投票筹码**: 扑克牌/游戏币/贴纸（人数 x 3）。
- [ ] **垃圾袋**: 大号黑色垃圾袋 1 卷。

### 🍫 食品与氛围类
- [ ] **饮用水**: 矿泉水 1-2 箱（或大桶水+纸杯）。
- [ ] **零食**: 独立包装的小饼干、糖果（补充脑力）。
- [ ] **奖品**:
    - 一等奖 (如社区 T 恤/马克杯)。
    - 二等奖 (如书籍)。
    - 三等奖 (如贴纸/徽章)。
- [ ] **奖状/证书**: 打印好的荣誉证书（留空名字）。



## 三、工作流程清单

### 📅 活动前 (T-3 ~ T-1)
- [ ] **采购**: 检查库存，缺啥买啥。
- [ ] **打包**: 将小件物品装入"物料箱"或行李箱，方便携带。
- [ ] **打印**: 打印签到表、桌贴、指引牌。

### ⏰ 活动当天
*   **12:00 - 13:30 (进场)**:
    - [ ] **分发**: 将插线板铺设到每个岛屿，用胶带固定。
    - [ ] **布置**: 在签到台摆放签到表、笔、姓名贴。
    - [ ] **茶歇**: 摆放水和零食区。
*   **活动中**:
    - [ ] **补给**: 观察水/纸巾是否够用，及时补充。
    - [ ] **回收**: 投票结束后回收筹码。
*   **18:00 (撤场)**:
    - [ ] **回收**: 收回所有插线板、转接头、麦克风（这是最贵的，别丢！）。
    - [ ] **清点**: 确认物资无丢失，记录消耗量（用了多少水、多少贴纸）。
    - [ ] **归位**: 将物资装箱带回。



## 四、常见问题与应急处理

| 问题 | 解决方案 |
|------|---------|
| **物料不够（如水、纸巾）** | 应急采购：附近便利店/超市；或向场地方借用 |
| **贵重物品丢失（如转接头、翻页笔）** | 立即在活动群发布寻物启事；检查签到表确认可能遗失者 |
| **插线板不够/跳闸** | 向场地方借用；或引导参与者轮流充电 |
| **奖品数量不匹配** | 提前与主理人确认获奖名额；准备通用奖品（如贴纸）作为备选 |



## 五、物料员锦囊
1.  **胶带是神器**: 带一卷宽胶带（封箱胶）和一卷美纹纸（不留胶），现场固定线缆、贴海报全靠它。
2.  **急救包**: 备一点创可贴、润喉糖，会有奇效。
3.  **私人物品**: 提醒大家保管好贵重物品，但如果有人遗失了充电器/水杯，帮忙收好并发群寻找失主。
