# 🍕 餐饮与后勤扩展 SOP (Catering & Logistics)


## 一、新增职责 (区别于小型活动)
*   **餐饮管理 (Catering)**：80人的披萨/茶歇是一个巨大的工程，需要精确计算和分发。
*   **动线规划 (Flow)**：人多了，哪里排队签到、哪里领饭、垃圾丢哪里，都需要设计。
*   **垃圾分类 (Waste)**：几百个披萨盒如果不及时处理，现场会变成垃圾场。



## 二、工作流程清单

### 🍕 订餐与分发
- [ ] **预算计算**: 确认人均标准（如 30元/人）。
- [ ] **预订**: 提前 1 天预订。
    - **避坑**: 披萨/炸鸡通常送得慢，要打好提前量（比如要求 11:30 送到，这样 12:00 能吃上）。
    - **口味**: 遵循"大众化"原则，避免太辣或太奇怪的口味。务必点 10% 的素食。
- [ ] **分发**:
    - 让大家自取！容易乱，需要注意
    - 设置"领餐台"，排队领取。
    - 或者由志愿者分发到每个小组的桌子上。

### 🥤 茶歇台维护
- [ ] **无限续杯**: 检查桶装水/咖啡/茶包是否充足。
- [ ] **去库存**: 观察哪些零食没人吃，下次别买了；哪些被秒光，下次多买点。

### 🧹 垃圾风暴
- [ ] **定时清理**: 每隔 1 小时巡视一次垃圾桶，满了立刻换袋子。
- [ ] **大件垃圾**: 披萨盒体积大，要安排专人拆解压扁后再丢弃。
- [ ] **厕所指引**: 在显眼位置贴上厕所指引，并检查卫生纸是否充足（如果物业不管，我们得管）。



## 三、常见问题与应急处理

| 问题 | 解决方案 |
|------|---------|
| **餐饮送达延误** | 提前与商家确认送达时间；准备零食应急；必要时调整活动流程 |
| **食物过敏/不满** | 提前在群里询问忌口；现场提供多样化选择；准备素食/清淡选项 |
| **垃圾桶满溢** | 增加垃圾桶数量；安排专人定时清理；必要时联系物业支援 |
| **餐饮数量不足/过剩** | 不足时应急采购补充；过剩时鼓励志愿者打包带走 |



## 四、后勤锦囊
1.  **宁多勿少**: 水和纸巾永远不嫌多。
2.  **过敏源**: 活动开始前，在大屏幕提醒："餐食含有海鲜/花生，过敏者请注意。"
3.  **剩菜处理**: 活动结束如果剩了很多吃的，鼓励志愿者打包带走，拒绝浪费。
