# 📦 后勤/签到 SOP

> **核心定位**: 活动的"门面担当"，场地的"大管家"。
> **📝 使用提示**: 请将下方清单视为 Checklist，完成一项勾选一项。



## 一、角色关键职责
*   **接待 (Reception)**：签到核销，发放物料，引导入群。
*   **后勤 (Logistics)**：场地布置，物资管理，订餐/茶歇。
*   **清场 (Cleanup)**： **最重要的职责之一**，活动结束后还原场地。



## 二、工作流程清单

### 📅 活动前
- [ ] **指引优化**:
    - 拍摄/制作清晰的"如何到达会场"指引图或**视频**（建议从地铁口/标志性建筑如麦当劳开始录制）。
    - 标注电梯位置、楼层等关键信息。
    - 发到群里，并添加到报名确认短信中。
- [ ] **物料准备**:
    - 签到表 (打印版/电子码)。
    - **打印资料**: 提前打印活动流程、WiFi二维码、共学文档二维码等，签到时分发，减少现场困惑。
    - 贴纸 (姓名贴、身份贴：程序/设计/PM)。
    - 投票筹码/贴纸 (如有，每人3个)。
    - 笔、A4纸、排插（务必确认排插是否准备好！！）。

### ⏰ 活动当天
*   **12:00 - 13:30 (布置)**:
    - [ ] **桌椅**: 按照分组岛屿式摆放，确保动线通畅。
    - [ ] **电源**: 铺设排插，用胶带固定线缆（防绊倒）。
    - [ ] **指引**: 在路口/电梯口张贴指引牌 / 易拉宝 / 海报等。如果场地复杂，安排志愿者在关键路口接应。
    - [ ] **签到台**: 设置在入口显眼处。
*   **13:30 - 14:30 (签到)**:
    - [ ] **核销**: 确认报名信息。
    - [ ] **发放**: 给每人贴上姓名贴（写大一点！），分发打印好的活动指南。
    - [ ] **引导**: "请扫码入群，里面有 WiFi 密码和活动文档。"
*   **17:00 (投票)**:
    - [ ] 协助发放/回收投票筹码 或者 协助投票。
    - [ ] **计票**: 提前确认计票方式，避免耽误时间或漏票。
*   **18:00 (清场 - 必做!)**:
    - [ ] **垃圾**: 清理所有桌面的水瓶、纸屑，打包带走。
    - [ ] **复原**: 桌椅归位，排插收纳。
    - [ ] **遗失物**: 检查是否有遗留物品，拍照发群。



## 三、常见问题与应急处理

| 问题 | 解决方案 |
|------|---------|
| **签到系统崩溃** | 使用打印版签到表备份，手动核销 |
| **有人找不到场地** | 提前安排一位志愿者在楼下/地铁口接应，或通过微信/电话远程指引 |
| **物料不够（如姓名贴、笔）** | 向附近便利店应急采购，或向场地方借用 |
| **排插不够/跳闸** | 检查是否有大功率设备（如电热水壶），分散用电负荷 |
| **场地空调/灯光故障** | 立即联系场地方物业，同时告知主理人调整流程 |



## 四、后勤锦囊
1.  **微笑**: 你是参与者见到的第一个人，你的笑容决定了他们对社区的第一印象。
2.  **姓名贴**: 提醒大家把名字写大一点，方便社交。
3.  **指引**: 哪怕有图，很多人还是会迷路。如果可以，安排一位"人肉指引"在楼下接人是体验最好的。
