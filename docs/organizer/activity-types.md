# Part 3: 📚 活动类型与核心流程

## 分享型活动（Sharing）
- **目的**：经验输出、连接同好、促进交流。
- **建议流程**：签到/开场 → 主题分享 → Q&A → 交流合影 → 收尾。

## 迷你黑客松（Mini Hackathon）
- **目的**：短时间协作产出 MVP。
- **建议流程**：签到/开场 → 组队/选题 → 共学开发 → 作品提交 → Demo → 颁奖/致谢 → 清场。
