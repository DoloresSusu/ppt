# 👔 活动主理人工作 SOP


## 一、角色关键职责
*   **统筹**：策划活动主题，把控整体进度，确保人员/物资/场地到位。
*   **志愿者管理**：招募、分工、培训，现场调度补位并做好保障。
*   **决策**：处理突发状况，做最终决策人。
*   **连接**：连接社区资源、讲师嘉宾与参与者。
*   **文档与沟通**：建立志愿者筹备群与活动现场群，指定专人准备共学文档和 PPT。




## 二、工作流程清单

### 📅 活动前 (T-14 ~ T-1)
- [ ] **建群（重要）**:
    - **志愿者群**: 拉入所有志愿者，发布分工表，发红包。
    - **活动现场群**: 提前建好，生成二维码放进 PPT 和签到台。
- [ ] **报名与志愿者招募**:
    - [ ] **发布报名**: 渠道包括社区网站（如 HackathonWeekly.com）、Lu.ma、Devpost、活动行、飞书问卷（WayToAGI）等。
    - [ ] **志愿者转化**: 在报名链接上直接询问"你愿意来做志愿者吗？"；报名后主动联系意向志愿者，提前提供培训文档。
    - [ ] **确认分工**: 填写志愿者分工表，明确角色与到场时间。
    - [ ] **工作群**: 建立当期活动工作群，发布分工表和 SOP 文档。
    - [ ] **订餐**: 统计志愿者人数与忌口（素食/过敏），预订工作餐/奶茶。
- [ ] **内容把控**:
    - **优先使用飞书文档/在线文档替代 PPT**: 可直接转换成活动推送和现场资料，支持多人在线编辑，避免手动制作 PPT 的时间成本。
    - 确认 **分享嘉宾/讲师** 的 PPT 和 共学文档 已准备好。
    - *PPT 模板*: [通用 PPT 模板](https://hackathonweekly.feishu.cn/wiki/M5B4w2j8iiKJzbksKX6chZBlnEd)（仅用于大型黑客松或必须时）
    - *共学文档模板*: [共学文档模板](https://hackathonweekly.feishu.cn/wiki/MUTFwqPGbiI0aGkYB7ncjBSsnog)
- [ ] **场地**:
    - 周周黑客松有常见的场地合作方，见 https://space.hackathonweekly.com
    - **选址建议**: 党群服务中心、创业孵化器、高校教室（成本低、氛围好）。
    - **确认设施**: 投影/音响/WiFi/插座/空调。
- [ ] **宣发**: 安排宣发组发布推文、海报。
- [ ] **物料**: 准备贴纸、奖状、零食等。

### ⏰ 活动当天 (T-0)
*   **12:00 (提前2小时到场)**:
    *   **原因**: 考虑志愿者可能迟到、参与者可能早到、留足设备调试和团队交流时间。
    *   点名确认志愿者到齐，缺席及时联系或安排替补。
    *   组织岗前培训（5-10 分钟）：快速过流程，强调签到引导与群内信息。
    *   带领志愿者进场，进行场地布置、设备调试、流程彩排。
*   **13:30 (确认)**: 确认所有岗位就绪（签到、摄影、技术、计时）。
*   **14:00 (开场)**: 配合主持人开场，必要时进行简短致辞。
*   **活动中**:
    *   **记录**: 强烈建议开启录音功能（飞书妙记/豆包/讯飞/腾讯会议），方便后续 AI 复盘。
    - [ ] **巡场**: 观察现场氛围，发现落单人员，协助主持人处理冷场。
    - [ ] **调度**: 发现岗位缺人及时补位，安排志愿者轮换休息。
    - [ ] **保障**: 确保志愿者有水喝、有饭吃，提醒适时休息。
    - [ ] **应急**: 处理断网、设备故障、时间延误等突发问题。
    - [ ] **挖掘**: 观察表现突出的参与者或志愿者，活动后发展为核心成员。
*   **17:30 (致谢)**: **重要环节**！上台为志愿者颁奖/致谢，按照"鼓励师"提供的细节进行夸奖。

### 🏁 活动后 (T+1)
- [ ] **合影**: 组织志愿者单独大合影，便于后续传播与纪念。
- [ ] **聚餐（深度链接）**: 尽量安排志愿者/核心成员饭局（AA制即可），这是社区建设的核心路径（参与活动 -> 聚餐 -> 建立信任 -> 成为下次志愿者/组织者）。
- [ ] **复盘**:
    - 立即利用活动录音，使用 AI 生成复盘报告。
    - 组织志愿者进行简单复盘，记录亮点与改进点。
- [ ] **归档**: 收集照片/视频，更新社区资产库。
- [ ] **财务**: 处理报销事宜（如有）。



## 三、常见问题与应急处理

| 问题 | 解决方案 |
|------|---------|
| **志愿者临时缺席** | 启动备用方案：自己顶上，或请其他志愿者兼任；提前准备每个关键岗位的备用人选 |
| **志愿者不熟悉工作内容** | 现场 5 分钟培训；安排经验志愿者带领 |
| **志愿者体力不支** | 安排轮换休息，提供饮水与座位 |
| **活动严重超时** | 与主持人、计时员协商，压缩非核心环节（如缩短问答），确保核心环节完成 |
| **现场秩序混乱** | 暂停流程，主持人重申规则；必要时请鼓励师/顾问协助稳定氛围 |
| **参与者投诉或冲突** | 立即私下沟通了解情况，必要时请鼓励师/顾问协助调解 |
| **赞助商临时要求** | 在不影响活动核心的前提下协商解决；如有冲突，礼貌说明情况并提供替代方案 |



## 四、主理人锦囊
1.  **放权**: 不要事必躬亲，相信志愿者，让他们在岗位上发光。
2.  **补位**: 你是最大的替补，哪里缺人哪里顶上，但首要任务是"统筹"。
3.  **关怀**: 志愿者也需要被看见与休息，一句"辛苦了"很有力量。
4.  **清晰指令**: 不要只说"去帮忙"，明确指派任务与负责人。
5.  **预案**: 关键岗位至少准备 1 名备选人员。
