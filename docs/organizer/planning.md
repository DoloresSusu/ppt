# 📝 活动策划与 AI 助手

为了让策划过程更轻松，我们推荐先使用 AI 助手生成初步方案，然后再使用标准模板进行微调。


## 第一步：🤖 AI 智能辅助筹备

复制下方的 AI 提示词，粘贴到 Claude、ChatGPT 或豆包中。它会引导你完成活动信息的收集，并帮你生成初步的策划文档。

### 📋 AI 提示词模板（点击复制）

```text
你是一位经验丰富的社区活动策划专家。请基于以下步骤，通过提问引导我完成一场社区活动的 SOP 定制。

## 第一步：了解活动基本信息
请依次询问我以下问题（每次问 1-2 个，等我回答后再继续）：
1. 活动名称是什么？
2. 活动类型？（分享型 / 迷你黑客松 / 工作坊 / 大型黑客松 / 其他）
3. 预计参与人数？
4. 活动日期和时间？（格式：2024年X月X日，HH:MM-HH:MM）
5. 活动地点？（详细地址 + 交通指引）
6. 活动主题是什么？（一句话描述）

## 第二步：角色招募与分配
根据活动类型，推荐必需的志愿者角色清单，并询问：
1. 各角色是否已确认人选？（已确认/待招募）
2. 如已确认，请告诉我姓名和联系方式

## 第三步：时间轴规划
根据活动类型和日期，生成时间轴清单，并询问：
1. 是否需要调整某些时间节点？
2. 是否有特殊环节需要加入时间轴？

## 第四步：物料与预算
询问：
1. 本次活动预算范围？
2. 是否有赞助商？需要什么权益露出？
3. 特殊物料需求？（如奖品、餐饮、设备）

## 第五步：特殊说明
询问：
1. 本次活动有什么特殊注意事项？
2. 需要重点强调给志愿者的内容？

## 最后：生成填写好的模板
根据我的回答，帮我生成一个完整的活动策划文档。

开始吧！请先问我第一个问题。
```


## 第二步：📝 完善活动信息模板

在 AI 帮你生成初步方案后，请将其填入下方的标准模板中。你可以将此内容复制到飞书文档或 Notion 中作为本次活动的“作战地图”。

### 活动信息模板

```md
# 📝 本次活动策划案

> **📝 填写说明**
> - 🔲 表示：待填写的空白项 | ✅ 表示：已确认
> - 【待确认】表示：信息尚未最终确定 | 【待招募】表示：角色空缺

## 📌 1. 基本信息
| 项目 | 内容 |
|------|------|
| **活动名称** | 🔲 _______________ |
| **活动类型** | 🔲 分享型 / 迷你黑客松 / 工作坊 / 其他：___ |
| **活动主题** | 🔲 _______________ |
| **日期时间** | 🔲 ___年___月___日，___:___-___:___ |
| **活动地点** | 🔲 _______________ |
| **交通指引** | 🔲 地铁___站___号口 / [链接] |
| **预计人数** | 🔲 ___人 |
| **预算/赞助** | 🔲 ___元 / 赞助商：___ |

## 👥 2. 核心分工
| 角色 | 姓名 | 到场时间 | 状态 |
|------|------|---------|------|
| **活动主理人** | 🔲 ___ | 提前 2h | ✅ |
| **主持人** | 🔲 ___ | 提前 1.5h | 【待招募】 |
| **后勤/签到** | 🔲 ___ | 提前 1.5h | 【待招募】 |
| **技术支持** | 🔲 ___ | 提前 1.5h | 【待招募】 |
| **摄影/宣发** | 🔲 ___ | 提前 1.5h | 【待招募】 |

## 📅 3. 活动时间轴
- [ ] **T-7天**：确定主题与场地，发布报名
- [ ] **T-3天**：物料准备完毕（贴纸、零食等）
- [ ] **T-1天**：志愿者线上同步会
- [ ] **活动当天 13:00**：志愿者到场布置
- [ ] **活动当天 14:00**：正式开始
- [ ] **活动当天 18:00**：活动结束、清场、聚餐

## 🎁 4. 物料清单
- [ ] **设备类**：投影仪、转接头、排插、音响
- [ ] **签到类**：签到二维码、名牌/贴纸、马克笔
- [ ] **餐饮类**：饮用水、零食、下午茶（如有）
- [ ] **氛围类**：社区横幅、周边贴纸、奖品
```
