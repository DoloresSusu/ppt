## 志愿者须知 {#volunteer-notices}

作为周周黑客松的志愿者，我们需要共同遵守以下准则：

1.  **准时到达**：请务必按照规定的集合时间到达场地。
2.  **主动协助**：在完成自己岗位工作之余，主动观察现场是否有需要帮忙的地方。
3.  **保持热情**：我们是一个温暖的社区，请微笑面对每一位参与者。
4.  **熟悉 SOP**：在活动开始前，请至少完整阅读一遍你的角色 SOP。
5.  **安全第一**：遇到任何突发状况或安全问题，第一时间反馈给活动主理人。
