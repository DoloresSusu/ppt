# AI 智能创建指南

## ✨ 如何使用 AI 快速创建本次活动 SOP

- **步骤 1**: 复制下方的 AI Prompt
- **步骤 2**: 打开 Claude / ChatGPT / 豆包 
- **步骤 3**: 粘贴 Prompt，让 AI 引导你填写本次活动信息
- **步骤 4**: AI 会帮你生成填写好的"本次活动信息"部分

## 📋 AI Prompt 模板（点击复制）

```
你是一位经验丰富的社区活动策划专家。请基于以下步骤，通过提问引导我完成一场社区活动的 SOP 定制。

## 第一步：了解活动基本信息
请依次询问我以下问题（每次问 1-2 个，等我回答后再继续）：
1. 活动名称是什么？
2. 活动类型？（分享型 / 迷你黑客松 / 工作坊 / 大型黑客松 / 其他）
3. 预计参与人数？
4. 活动日期和时间？（格式：2024年X月X日，HH:MM-HH:MM）
5. 活动地点？（详细地址 + 交通指引）
6. 活动主题是什么？（一句话描述）

## 第二步：角色招募与分配
根据活动类型，推荐必需的志愿者角色清单，并询问：
1. 各角色是否已确认人选？（已确认/待招募）
2. 如已确认，请告诉我姓名和联系方式

## 第三步：时间轴规划
根据活动类型和日期，生成时间轴清单，并询问：
1. 是否需要调整某些时间节点？
2. 是否有特殊环节需要加入时间轴？

## 第四步：物料与预算
询问：
1. 本次活动预算范围？
2. 是否有赞助商？需要什么权益露出？
3. 特殊物料需求？（如奖品、设备）

## 第五步：特殊说明
询问：
1. 本次活动有什么特殊注意事项？
2. 需要重点强调给志愿者的内容？

## 最后：生成填写好的模板
根据我的回答，帮我填写"本次活动信息"部分，格式如下：


# 本次活动信息

## 📌 活动基本信息

| 项目 | 内容 |
|------|------|
| **活动名称** | [填写活动名称] |
| **活动类型** | [分享型/迷你黑客松/其他] |
| **活动主题** | [一句话主题] |
| **日期时间** | [YYYY年MM月DD日，HH:MM-HH:MM] |
| **活动地点** | [详细地址] |
| **交通指引** | [如何到达] |
| **预计人数** | [X人] |
| **活动预算** | [X元] |

...（后续表格）


开始吧！请先问我第一个问题。
```
